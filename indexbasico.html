QUESTIONARIO

Questao 01: Explique qual é a função de um servidor web em um ambiente de hospedagem e cite pelo menos um exemplo.
R= O servidor web é responsável por receber as requisições dos navegadores, processá-las e entregar o conteúdo do site (como páginas HTML e imagens). Ele garante que o site seja acessado corretamente pelos usuários. Um exemplo popular de servidor web é o Apache HTTP Server.
Questao 02: Qual a principal característica do NGINX e por que ele é muito utilizado em projetos de alto desempenho?
R= A principal característica do NGINX é sua arquitetura assíncrona, que permite lidar com muitas conexões simultâneas de maneira eficiente. Ele é muito utilizado em projetos de alto desempenho por ser leve, rápido e ideal para balanceamento de carga e serviço de conteúdo estático.
Questao 03: Diferencie os protocolos HTTP e HTTPS. Por que o HTTPS é recomendado para aplicações modernas?
R= HTTP não tem criptografia, já HTTPS usa SSL/TLS para segurança. O HTTPS é recomendado para proteger dados sensíveis.
Questao 04: Qual é a função das portas em uma rede de computadores e qual a importância da porta 443?
R= Portas identificam serviços na rede. A porta 443 é usada para HTTPS, garantindo comunicação segura.
Questao 05: Defina o conceito de endereço IP e explique sua importância em redes de computadores.
R= O IP é o identificador único de um dispositivo na rede, essencial para comunicação entre eles.
Questao 06: Explique o que é hospedagem VPS e em quais situações ela é mais indicada.
R= VPS oferece recursos dedicados em um servidor compartilhado, ideal para sites com tráfego alto e necessidade de personalização.
Questao 07: Descreva as vantagens do servidor web Caddy em relação a outros servidores tradicionais.
R= O Caddy facilita a configuração de HTTPS e é simples de usar, com recursos de balanceamento de carga integrados.
Questao 08: Qual servidor de aplicação é mais indicado para aplicações desenvolvidas na plataforma .NET (C#)? Explique o motivo.
R= O IIS é o servidor ideal para .NET, pois é integrado ao Windows e otimiza a execução de aplicações ASP.NET.
Questao 09: Descreva o funcionamento do modelo Cliente-Servidor no contexto de sistemas web.
R= No modelo Cliente-Servidor, o cliente solicita e o servidor responde com conteúdo ou dados processados.
Questao 10: Explique o que é o Apache Tomcat e para qual tipo de aplicação ele é mais indicado.
R=O Tomcat é um servidor de aplicação Java, ideal para Java Servlets e JSP.
Questao 11: Diferencie, com suas palavras, um servidor web de um servidor de aplicação, destacando suas funções principais.
R= Servidor web serve conteúdo estático, enquanto servidor de aplicação processa lógica e gera conteúdo dinâmico.
Questao 12:Defina o conceito de domínio na internet e sua relação com os endereços IP.
R= Um domínio é um nome legível, mapeado para um endereço IP que identifica o servidor na rede.
Questao 13: Quais as principais vantagens da hospedagem em nuvem em comparação com a hospedagem local?
R= A nuvem oferece escalabilidade e flexibilidade, enquanto a local exige investimento em infraestrutura própria.
Questao 14: Cite dois exemplos de servidores de aplicação e as linguagens/plataformas que eles suportam.
R= Tomcat (Java), Node.js (JavaScript).
Questao 15: Por que é essencial a utilização de HTTPS em sites e sistemas atualmente? Quais riscos o HTTP apresenta?
R=HTTPS protege dados sensíveis e evita ataques. HTTP é vulnerável a interceptações.
Questao 16: Dê um exemplo de situação onde o uso de hospedagem VPS seria mais vantajoso do que uma hospedagem compartilhada.
R= VPS oferece mais controle e desempenho, ideal para sites com alto tráfego.
Questao 17: Diferencie os conceitos de IP e porta em uma rede, explicando suas funções.
R= IP identifica o dispositivo; porta define o serviço que processa a comunicação.
Questao 18: Qual a importância de um servidor web conseguir lidar com conexões via HTTPS?

Questao 19: Cite dois exemplos de servidores web e explique suas principais características.
R= Apache: flexível, muitos módulos. NGINX: rápido, leve, ideal para alto desempenho.
Questao 20: Defina o que é uma hospedagem compartilhada e em quais casos ela costuma ser utilizada.
R= Servidores compartilhados com vários sites. Ideal para sites pequenos e com baixo tráfego.
